这是我的第一个制作的游戏
🎮 屠龙勇士

一款使用Godot 4.6引擎开发的2D平台动作游戏。玩家需要控制角色躲避绿色和红色的粘液怪物，
在充满挑战的平台世界中生存和冒险。

🎯 游戏特性

🕹️ 流畅的角色控制
- 多段跳跃：支持三段跳跃，灵活躲避敌人
- 翻滚冲刺：按下C键执行高速翻滚，有爆发速度和摩擦力系统
- 自然物理：真实的重力系统和碰撞检测

👾 智能敌人系统
- 粘液怪物：绿色和红色两种敌人，有独立的AI行为
- 智能巡逻：通过射线检测自动感知墙壁和障碍，改变移动方向
- 碰撞反应：与玩家接触时的实时互动

🎨 精美的视觉效果
- 流畅动画：角色和敌人的多帧动画过渡
- 场景设计：多层平台、安全区和危险区相结合
- 视觉反馈：通过杀伤区的设置让玩家清楚地了解危险范围

🎮 游戏操作

| 按键 | 功能 |
|------|------|
| A / ←  | 向左移动 |
| D / → | 向右移动 |
| Space | 跳跃（可连续按下实现多段跳） |
| C | 翻滚冲刺 |

## 🛠️ 技术栈

- 引擎: Godot 4.6
- 语言: GDScript
- 平台: Windows / Linux / macOS / Web
- 核心系统: 
  - 状态机控制系统
  - 2D物理引擎
  - 动画系统
  - 碰撞检测系统
